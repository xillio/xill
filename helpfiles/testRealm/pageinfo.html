pageinfo

pageinfo(<node,download,xml> page)

Description

Returns a list variable with specific info on the page object page such as: cookies, encoding, load time, title, status code, etc.

Examples

  • Executing the following code:

    webclient({"enablecss"=false, "enablejs"=false});
homeurl = "http://www.xillio.com/";
page = loadpage (homeurl);
log (pageinfo(page));

  • will print:

    {
    url = "http://www.xillio.com/",
    title = "Xillio",
    encoding = "UTF-8",
    contenttype = "text/html",
    loadtime = 275,
    requesturl = "http://www.xillio.com/",
    statuscode = 200,
    statusmessage = "OK",
    cookies = {}
}
