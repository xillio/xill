loadpage

loadpage(string url)

Description

Loads the webpage found at url and returns it as a page object. If loading fails the page object will contain an error code, which can be accessed using the function geterror. Also note that it is possible and common to read files by calling loadpage with a url starting with file:// as the parameter. Usually it is a good idea to reduce loading time by setting up the webclient({"enablecss"=false, "enablejs"=false}) before calling loadpage. This deactivates Javascript and CSS and dramatically reduces long waits for script timeouts.

Examples

  • Loop through menu on a webpage and store all navigation items:

    webclient({"enablecss"=false, "enablejs"=false});

  • homeurl = "http://www.xillio.com/";

  • page = loadpage (homeurl);
    mainmenutitles = xpath (page, "//ul[@class='active-branch']/li/a/text()");
mainmenuurls = xpath (page, "//ul[@class='active-branch']/li/a/@href");

  • foreach (numberinlevel, mainmenuurl, mainmenuurls) {
        log(mainmenutitles[numberinlevel] + " (" + mainmenuurls[numberinlevel] + ")");
}
