webclient

webclient(list options)

Description

Optional call to initialise a new webclient, overriding default options which will be used in calls to functions like loadpage() and download().DetailsThe argument options is a list which may contain one or more of the following values:optionvaluesbrowser[INTERNET_EXPLORER_8, INTERNET_EXPLORER_7, INTERNET_EXPLORER_6, FIREFOX, CHROME] (default: FIREFOX)proxyhost<hostname or ip> (required for using a proxy)proxyport<hostport> (required for using a proxy)proxyuser<username> (optional)proxypass<username> (optional)proxydomain<domainname> (required for NT authentication)proxyworkstation<workstationname>( required for NT authentication)enablecss[true,false] (default: true)enablejs[true,false] (default: true)enablecookies[true,false] (default: true)enableredirect[true,false] (default: true)insecuressl[true,false] (default:false)user<username> (required for HTTP authentication)pass<username> (required for HTTP authentication)timeout<timeoutvalue> (in milliseconds)urlfilter<urllist> (list of strings containing URL’s)loaderrorpages[true,false] (default: false)jstimeout<timeoutvalue> (Timeout in milliseconds after pageload, before javascripts running on pages are killed, DEFAULT: 0)

Examples

  • Most used call to webclient() (deactivate CSS and JS for faster loading of pages):

    webclient({"enablecss"=false, "enablejs"=false});

  • Loging in via HTTP authentication to a simple Sharepoint environment:

    webclient({proxyuser=settings.user, proxypass=settings.pass, proxyhost=settings.host, proxyport="80", proxydomain=settings.domain, proxyworkstation=""});

  • Please note the empty string for proxyworkstation.
