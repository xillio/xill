urlencode

urlencode(<string,void> text)

Description

Returns the URL-encoded (percent-encoded) version of string text. Essential for creating URL’s with GET variables in them. Note that spaces are converted to plus signs.

Examples

  • Executing:

    log (urlencode("Together, we spent $50.95"));

  • will print:

    Together%2C+we+spent+%2450.95
