es_regexfilter

es_regexfilter(string field,, string regex)

Description

Matches the value of field against regex. This returns a string identifier that can be used to identify this filter. If the document does not have field it will not be included in the search query.For more information on the regex syntax see: http://www.elastic.co/guide/en/elasticsearch/reference/current/query-dsl-regexp-query.html#regexp-syntax

Examples

  • Find the documents which have the words “color”, “colour”, “colored” or “coloured” in the field “text”:

    es_connect();
//This regex filter will match "color", "colour", "colored" and "coloured".
regexfilter = es_regexfilter("text", "colou?r(ed)?");
result = es_search("art", "photos", regexfilter);
