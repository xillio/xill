continue

continue()

Description

Skips directly to the next iteration of the current loop (while, foreach).

Examples

  • This will output the numbers 1 to 5.

    items = {1,2,3,4,5,6,7,8,9};
foreach (_, item, items) {
  if(item >= 5){
    continue();
  }
  log(item);
}
