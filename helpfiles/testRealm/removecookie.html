removecookie

removecookie(<void,string,list,boolean> cookie)

Description

Removes the browser cookie with the name cookie. cookie can be a string or a list. If cookie is a boolean with value true, then all cookies will be cleared. Current cookie information can be obtained through pageinfo(). Cookies can be set using setcookie().Due to the nature of Content Tools, all cookies will be removed with every restart.

Examples

  • Remove all cookies, set a cookie for the current page and remove it again:

    webclient({"enablecss"=false, "enablejs"=false});
homeurl = "http://www.xillio.com/";
page = loadpage (homeurl);
removecookie(true);
cookie = {name="Xillio", path="/", domain="www.xillio.com", value="123", expires="2013-10-18T09:42:15"};
setcookie(cookie);
log (pageinfo (page));
removecookie("Xillio");
log (pageinfo (page));
