ampersanddecode

ampersanddecode(<void,string> text,, number passes )

Description

Decodes all ampersand-encoded characters in the provided string. Mainly used to decode XML characters to their original characters. Optionally you can provide the number of passes the decoder should run.

Examples

  • Executing:

    ampersanddecode("Jack &amp; Jill");

  • Returns:

    "Jack & Jill"

  • Executing:

    ampersanddecode("Money &lt;&amp;gt; Health", 2);

  • Returns:

    "Money <> Health"
