getframe

getframe(node parentpage,, string frame)

Description

This browser function returns the contents of a named HTML frame frame as a page. Useful for loading an iframe or normal content frame.

Examples

  • Load the contents of an iframe into the variable frame:

    webclient({"enablecss"=false, "enablejs"=false});

  • page = loadpage ("http://www.w3schools.com/html/html_iframe.asp");

    frame = null;

  • if (page != null) {

        frame = xpath (page, "//iframe[not(@id) and not(contains(@src, 'facebook')) and not(contains(@class, 'goog'))]/@src");
    frame = loadpage (absoluteurl ("http://www.w3schools.com/html/", frame));
} else {
    log ("no page");
}
