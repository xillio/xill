focus

focus(node element,, boolean focus)

Description

This browser function sets or unset the focus to the node element. Set focus to true to give the element focus, false to release the focus. This is used as a replacement for mouseover and mouseout events in the absence of a mouse. Some pages require these events to be triggered to load data for combo-boxes, menus or other dynamic elements.

Examples

  • The sets focus to the search box in the upper right corner of the homepage, triggering any events that are associated with it (such as loading a history of search terms):

    webclient({"enablecss"=false, "enablejs"=false});

  • page = loadpage ("http://www.xillio.com/");

  • if (page != null) {
        node = xpath (page, "//input[@name='q']"); // select the search box at the top of the Xillio homepage
    focus (node, true); // set focus to that box
} else {
    log ("no page");
}
