getobject

getobject(string table,, list object,, <string,void> connectionstring )

Description

Returns a database row as a list object. Selects from database table table, based on any columns that have been set in object. The optional parameter connectionstring is a string identifying the database connection as returned by the database function. If no connection is given then the last used database connection will be used.

Examples

  • Query a metadata table based on pageid (primary key), change the value of a field and store it again:

    metadata = getobject ("metadata", {"pageid" = pageobject.pageid});
metadata.organisation = "xillio";
storeobject("metadata", metadata, "pageid");

  • Use a multi column primary key:

    storeobject(settings.db_importtable, import, {xillioid = "xillioid", type = "type"}, true);
