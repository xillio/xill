setprogress

setprogress(number progress,, number total ,, string message )

Description

Passes a progress indication to the system so that it can be displayed to the enduser through the user interface.The option progress should be a number between 0 and total. Total defaults to 1. If progress is larger than total, or if either option is negative, a progress of 100% will be set.Optionally, a message can be provided that will be displayed in front of the progress bar.

Examples

  • Set progress to 50%.

    setprogress(0.5);

  • Set progress to 5 out of 13.

    setprogress(5, 13);
