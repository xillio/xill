listfolders

listfolders(string uri,, boolean recursive ,, boolean showaccess )

Description

Returns an iterable containing the names of all subfolders in the folder specified by uri. Returns empty if uri is empty or not a folder.If the optional parameter recursive is set to true, it will also return all sub-subfolders.The function always returns even the folders that are not accessible (but visible) for current user. If optional parameter showaccess is set to true (default is false), the list of pairs is returned. Each pair contains key which is the name of the directory and the boolean value which means if the folder is accessible or not.

Examples

  • Display a list of all subdirectories of C:/tmp/, including their own subdirectories:

    folders = listfolders ("C:/tmp/", true);
foreach (_, folder, folders) {
    log (folder);
}

  • There is a nice trick for getting the number of folders in the list:

    folders = listfolders ("C:/tmp/"); // the files variable now contains the folder list
number = length(folders);
log (number);

  • This will print the number of subdirectories in the given directory.
