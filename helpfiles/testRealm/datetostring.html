datetostring

datetostring(date date,, string format,, <string,void> timezone)

Description

Returns the date as a string formatted as given in format, with respect to the timezone.Detailstimezone can be either an abbreviation such as “PST”, a full name such as “America/Los_Angeles”, or a custom zone such as “GMT-8:00”.If the timezone cannot be understood, “GMT” is used.SymbolMeaningType
