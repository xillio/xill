getcell

getcell(sheet sheet,, <string,number> column,, <string,number> row,, <boolean,void> evaluate )

Description

Retrieves the contents of cell column, row from Excel spreadsheet sheet. column and row can either be an index number (1,5) , or a readable indicator (A, 5). row can be omitted by putting the full cell indicator in the second parameter (A5). Note that column and row in numeric form are 1-based (counting starts at 1) to make them compatible with string-based lookups. When evaluate is set to false, formulas in the sheet will not be evaluated but return null instead.

Examples

  • Logging all column values in a spreadsheet:

    workbook = openexcel("C:/projects/someproject/mapping/mapping.xls");
sheet = getsheet(workbook, "Mapping");
numberRows = sheet["rows"];
numberColumns = sheet["columns"];
rowCounter = 1;
while (rowCounter <= numberRows)
{
        colCounter = 1;
        while (colCounter <= numberColumns)
        {
                log(rowCounter + "," + colCounter + ": " + getcell(sheet, colCounter, rowCounter));
                colCounter = colCounter+1;
        }
        rowCounter = rowCounter+1;
}

  • Possible uses of getcell:

    cell = getcell(sheet, "A1");
cell = getcell(sheet, "A", 1);
cell = getcell(sheet, "A", "1");
cell = getcell(sheet, 1, 1);
