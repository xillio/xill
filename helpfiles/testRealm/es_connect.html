es_connect

es_connect(<string,void> host ,, <number,void> port ,, <number,void> cluster )

Description

Connects to a running instance of elasticsearch. This returns a string identifier that can be used with other elasticsearch functions to identify this connection.The first connection will be remembered, and all other functions will use the first connection unless you explicitly tell them to use a different one (by passing the appropriate connection as a parameter). All parameters are optional, see the default values below.Default valueshost:localhostport:9300cluster:elasticsearch

Examples

  • Common usage:

    settings = global(callbot("settings.sbot"));
es_connect(settings.es_host, settings.es_port, settings.es_cluster);

  • To connect using the defaults use:

    es_connect();
