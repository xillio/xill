sqlescape

sqlescape(string value)

Description

Returns an SQL-escaped version of string value. This is essential for sending a webpage or unchecked input to an XML database.

Examples

  • To prepare the XML-file note.xml:

    <note id="1">
    <to>Tove</to>
    <from>Jani</from>
    <heading>Reminder</heading>
    <comment>This was sent to James as well</comment>
    <comment>John has not responded to the invitation</comment>
    <body>Don't forget me this weekend!</body>
</note>

  • for sending it to a database one might use:

    xml = loaddata ("c:/tmp/note.xml");
sqlstring = sqlescape (xml);
query ("INSERT INTO `xmltable` VALUES(" + sqlstring + ")");

  • Now sqlstring will contain:

    <note id=\"1\">\r\n    <to>Tove</to>\r\n    <from>Jani</from>\r\n    <heading>Reminder</heading>\r\n    <comment>This was sent to James as well</comment>\r\n    <comment>John has not responded to the invitation</comment>\r\n    <body>Don\'t forget me this weekend!</body>\r\n</note>
