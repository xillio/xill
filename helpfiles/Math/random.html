random

Math.random( value )

Description

Returns a randomized value based on the optional parameter value. If an integer argument N is provided the return value is a random integer between 0 and N. If a list argument L is provided it is a random entry from the list. If no parameter is provided, a random floating point number between 0 and 1 will be generated.

Examples

  • Roll a 6-sided die three times and print the results:

        System.print(Math.random (6)+1)
    
	Note: the + 1 is needed, because the result of random is 0 to 6, but not including 6.

  • Determine a random work day and print it:

        list = {"monday", "tuesday", "wednesday", "thursday", "friday"};
	System.print(Math.random((list))

  • Usage without parameter:

        log (random ());

  • prints a number like:

        0.11632623660157837
