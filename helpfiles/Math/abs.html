abs

Math.abs( 0 value )

Description

Returns the absolute value of the provided value.

Examples

  • Perform abs on three different inputs:

        System.print(Math.abs(6))
    System.print(Math.abs(-5))
    System.print(Math.abs(-3.14))

  • This will print:

        6
    -5
    -3.14
