Math

abs

abs( number value )

Returns the absolute value of the provided value.

hungarianalgorithm

hungarianalgorithm(list matrix,, boolean max )

Solves the optimal assignment problem using the Hungarian Algorithm on matrix, which is a two-dimensional list containing numbers. Returns the optimal combination of rows / columns and the optimal sum as an object. The optional argument max (defaults to true) defines wether maximum sum or minimum sum should be returned. Useful for calculating optimal costs or income based on a matrix.

random

random(<void,number,list> value )

Returns a randomized value based on the optional parameter value. If an integer argument N is provided the return value is a random integer between 0 and N. If a list argument L is provided it is a random entry from the list. If no parameter is provided, a random floating point number between 0 and 1 will be generated.

round

round(number value )

Rounds the provided value to the neares whole number.
