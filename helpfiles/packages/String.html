String

absoluteurl

absoluteurl( pageurl,) relativeurl)

Converts the relative URL string relativeurl to an absolute url using the string pageurl as base URL. Returns the resulting URL as a string.

base64decode

base64decode( content,) filename)

Base-64 decode the string content, and store it in the file filename. Commonly used to decode binary data transferred over a text channel (such as attachments in e-mails).

base64encode

base64encode( file)

Returns the base-64 encoded string of file file. Commonly used to encode binary data for transfer over a text channel (such as attachments in e-mails). The parameter file is expected to be either a valid file name, or a binary variable.

ampersanddecode

ampersanddecode( text,) passes)

Decodes all ampersand-encoded characters in the provided string. Mainly used to decode XML characters to their original characters. Optionally you can provide the number of passes the decoder should run.

ampersandencode

ampersandencode( text)

Encodes all special XML characters (,>,&,”,’) to their respective xml entities.
