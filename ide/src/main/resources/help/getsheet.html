getsheet

getsheet (workbook workbook, <string,number> worksheet )

Description

Retrieves the indicated worksheet from the workbook. The result also provides shorthand information about the worksheet in the form of row count and column count.

Example

  • Fetch row and column count in a spreadsheet:

    workbook = openexcel("C:/projects/someproject/mapping/mapping.xls");
sheet = getsheet(workbook, "Mapping");
numberRows = sheet["rows"];
numberColumns = sheet["columns"];
