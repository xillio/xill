base64decode

base64decode (<void,string> content, string filename)

Description

Base-64 decode the string content, and store it in the file filename. Commonly used to decode binary data transferred over a text channel (such as attachments in e-mails).

See also base64encode

Example

  • base64decode("PG5vdGU+DQo8dG8+VG92ZTwvdG8+DQo8ZnJvbT5KYW5pPC9mcm9tPg0KPGhlYWRpbmc+UmVtaW5kZXI8L2hlYWRpbmc+DQo8Ym9keT5Eb24ndCBmb3JnZXQgbWUgdGhpcyB3ZWVrZW5kITwvYm9keT4NCjwvbm90ZT4=", "C:/tmp/test.txt"));

    returns null and decodes the encoded string to the textfile test.txt, which will contain the following text:

    <note>
<to>Tove</to>
<from>Jani</from>
<heading>Reminder</heading>
<body>Don't forget me this weekend!</body>
</note>
