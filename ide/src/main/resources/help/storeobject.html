storeobject

storeobject (string table, list object, <string,list> keys, boolean overwrite*, <string,void> connectionstring*)

Description

Stores the list variable object into an existing table named table. This will update existing rows if found, or insert a new row otherwise. keys is either a string containing the name of a single column in object or a list of multiple strings with the same key and value as a column name. The column or columns named by keys will be used as a primary key. The optional parameter overwrite is a boolean. If set to true then object will update an existing row in the database if found. If overwrite is set to false then an existing row will not be updated. The optional parameter connectionstring is a string identifying the database connection as returned by the database function. If no connection is given then the last used database connection will be used.

See also getobject

Example

  • Query a metadata table based on pageid (primary key), change the value of a field and store it again:

    metadata = getobject ("metadata", {"pageid" = pageobject.pageid});
metadata.organisation = "xillio";
storeobject("metadata", metadata, "pageid");
