-----------
mongo_store
-----------

**mongo_store (**\ *string* **collection,** *list* **object,** *string* **mongo*)**

Description
-----------

Stores ``object`` in the specified ``collection``, and returns the inserted object. If no "_id" is specified in the object, one will be generated by mongo. The returned object will then also contain the generated id. If `object` is an associative array (key = value) it will be treated as one document. Otherwise (if it is a list of arrays) it will be treated as a list of documents. If ``mongo`` is not provided the last active connection will be used.

See also :doc:`mongo_update`

Examples
--------

-

    Storing a new person in the "employees" collection:
    
    ::
    
        mongo_connect("localDB");
        john = { name = "John", last = "Doe", languages = { "C#", "Java", "Haskell" } };
        inserted = mongo_store("employees", john);
        log(inserted);

    Will print:
    
    ::
    
        {
            "_id": "507f1f77bcf86cd799439011",
            "name": "John",
            "last": "Doe",
            "languages": {
                "C#",
                "Java",
                "Haskell"
            }
        }
