----
help
----

**help (**\ *string* **topic)**

Description
-----------

Prints the helptext for the Xillio Content Tools language element (function or keyword) ``topic``. If ``topic`` is ``null``, ``help`` will print a complete list of available elements.

Examples
--------

-

    Returns a string of all language elements:
    
    ::
    
        help();
        
-

    Returns the help text for the ``xpath`` function:
    
    ::
    
        help("xpath");
