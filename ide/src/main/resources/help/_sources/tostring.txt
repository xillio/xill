--------
tostring
--------

**tostring (**\ *any* **value*)**

Description
-----------

Attempts to convert a variable into a string.

Examples
--------

-
    
    Append two numbers:
    
    ::
    
        log(tostring(5) + tostring(30));
        
    This will print:
    
    ::
    
        530
