--------------
es_indexexists
--------------

**es_indexexists (**\ *string* **index,** *string* **client*)**

Description
-----------

Returns ``true`` if the specified ``index`` exists on cluster ``client``, and ``false`` otherwise.

Examples
--------

-

	Check if the index "someindex" exists:

	::
	
		es_connect();
		exists = es_indexexists("someindex");
		if (exists) {
			//Perform actions
		}
