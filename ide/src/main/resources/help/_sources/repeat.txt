------
repeat
------

**repeat (**\ *string* **value,** *<number,void>* **count)**

Description
-----------

Returns a string containing the string ``value`` exactly ``count`` times.

Example
-------

-
    
    Executing:
    
    ::
    
        log(repeat("developers ", 5));

    will print:
    
    ::
    
        developers developers developers developers developers
