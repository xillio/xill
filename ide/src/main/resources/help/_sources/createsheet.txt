-----------
createsheet
-----------

**createsheet (**\ *workbook* **workbook,** *<string>* **name)**

Description
-----------

Creates a sheet with given ``name`` in provided ``workbook``. After the sheet has been created, cell values can be set by using :doc:`setcell`.

Example
-------

-
    
    Creates an empty sheet with name ``sheet``:
    
    ::

        workbook = createexcel();
        sheet = createsheet(workbook, "Mapping");
