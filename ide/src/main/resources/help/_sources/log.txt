log
---

**log (**\ *any* **text\*,** *string* **loglevel*)**


Description
-----------

Prints the contents of variable ``text`` to the log (currently always the console). ``text`` can be of any type, depending on the type the generated output will vary. Currently all logs will be preceded by the prefix ``"INFO: "``. Trying to log a ``null`` value has no effect.
The optional parameter ``loglevel`` may contain one of the following values: debug, info, warning, error. By default logging is sent to info.

Examples
--------

-
    
    ::
    
        log ("Starting...");
    
    will print:
    
    ::
    
        INFO: Starting...
    
-

    ::
    
        number = 150 / 3;
        log (number);
    
    will print:
    
    ::
    
        INFO: 50
    
-

    ::
    
        boolean = true;
        log (boolean, "warning");
        
    will print:
	
    ::
    
        WARN: true
        
-
    
    ::
    
        list = {id = 1, name = "Thomas", age = 12};
        log (list, "foo");
        
    will print:
    
    ::
    
        INFO : {
          "age": 12,
          "id": 1,
          "name": "Thomas"
        }

-
    
    ::
    
        page = loadpage ("http://www.xillio.com/");
        node = xpath (page, "//input[@name='q']");
        log (node);
    
    will print:
    
    ::
    
        INFO : HtmlTextInput[<input type="text" name="q">]
