----------
deletefile
----------

**deletefile (**\ *string* **uri)**

Description
-----------

Deletes the file located at ``uri``. If the file cannot be deleted an error is logged.


Example
-------

-

    Deletes the file ``test.txt``:
    
    ::
    
        deletefile ("C:/tmp/note.txt");
