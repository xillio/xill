------------
deletefolder
------------

**deletefolder (**\ *string* **uri)**

Description
-----------

Deletes the folder (and its contents) located at ``uri``. If the folder cannot be deleted an error is logged.


Example
-------

-

    Deletes the folder ``C:/tmp/123``:
    
    ::
    
        deletefolder ("C:/tmp/123");
