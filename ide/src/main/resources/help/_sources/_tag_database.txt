-----------
Database
-----------

Functions in this category perform database operations. After a connection is established with the :doc:`database` function then other options such as :doc:`query` can be executed. Content Tools automatically closes open connections when they're no longer used. The list of natively supported database platforms can be found inside the :doc:`database` function.
