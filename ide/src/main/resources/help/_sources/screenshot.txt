--------
screenshot
--------

**screenshot (**\ *node* **parentpage,** *string* **filename)**

Description
-----------

This browser function creates a screenshot of a current web page and save it to file. The file format is .png. 


Example
-------

-

    ::
 
        homeurl = "http://www.xillio.com/";
        page = loadpage(homeurl);
        screenshot(page, "c:/tmp/xillio-screenshot.png");
