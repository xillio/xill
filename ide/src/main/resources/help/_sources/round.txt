------
round
------

**round (**\ *number* **value*)**

Description
-----------

Rounds the provided value to the neares whole number.

Examples
--------

-   
    
    Perform round on three different inputs:
    
    ::
    
        log(round(6));
        log(round(3.14));
        log(round(-3.78));
		
	This will print 6, 3 and -4.
