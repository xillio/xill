listfiles

listfiles (string uri, boolean recursive*)

Description

Returns an iterable containing the absolute file names in the folder specified by uri. Returns empty if uri is empty or not a folder. If the optional parameter recursive is set to true, this function will also return the files in all subfolders of uri.

See also listfolders

Examples

  • Looping through all files in a directory (in this case stored webpages):

    cmsdirectory =  "C:/tmp/test/html-cms/";
cmspages = listfiles (cmsdirectory);

foreach(_, cmsfile, cmspages) {
    cmspage = trim (loaddata(cmsfile), true);
    if (cmspage != null) {
        log ("Successfully read page from: " + cmsfile);
    }
}

  • There is a nice trick for getting the number of files in a directory:

    files = listfiles ("C:/tmp/"); // the files variable now contains the file list
number = length(files);
log (number);

    This will print the number of files in the given directory. Subdirectories do not count as files.
