base64encode

base64encode (<string,binary> file)

Description

Returns the base-64 encoded string of file file. Commonly used to encode binary data for transfer over a text channel (such as attachments in e-mails). The parameter file is expected to be either a valid file name, or a binary variable.

See also base64decode

Example

  • base64encode("C:/tmp/note.xml");

    reads the textfile note.xml and returns

    "PG5vdGU+DQo8dG8+VG92ZTwvdG8+DQo8ZnJvbT5KYW5pPC9mcm9tPg0KPGhlYWRpbmc+UmVtaW5kZXI8L2hlYWRpbmc+DQo8Ym9keT5Eb24ndCBmb3JnZXQgbWUgdGhpcyB3ZWVrZW5kITwvYm9keT4NCjwvbm90ZT4="

    assuming note.xml contains the following text:

    <note>
<to>Tove</to>
<from>Jani</from>
<heading>Reminder</heading>
<body>Don't forget me this weekend!</body>
</note>
