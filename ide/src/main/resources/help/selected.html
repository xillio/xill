selected

selected (node element)

Description

The function returns true if the element node is selected. It can be used primarily at radio and check buttons.

See also select

Example

  • Assuming we have an HTML-file select.html containing the following:

    <html>
    <body>
        <select>
            <option value="volvo">Volvo</option>
            <option value="saab">Saab</option>
            <option value="mercedes">Mercedes</option>
            <option value="audi">Audi</option>
        </select>
    </body>
</html>

    then executing:

    page = loadpage("file:///C:/tmp/select.html");
selectbox = xpath(page, "//select");
choice = xpath(page, "//option[@value='saab']");
before = selected(choice);
select (choice, true);
after = selected(choice);
log(before);
log(after);

    will print:

    false
true
