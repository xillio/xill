date

date (<string,number> date*, string format*, string timezone*)

Description

Creates a date based on the provided arguments:

  • If no arguments are provided, it will return a date based on now()
  • If the first argument is a number, it will treat this number as a unix timestamp
  • If the first argument is a string, it will parse it based on the default format of yyyy-MM-dd’T’hh:mm:ss:S. An alternative format to be used can be passed as second parameter.
  • The timezone argument can be used to indicate the timezone that is used for the provided date. When no timezone is provided, ‘Europe/Amsterdam’ will be used. A list of allowed timezones values is provided below.

Examples

  • Executing:

    log(date());

    prints:

    2013-10-24T09:18:41.761

  • Executing:

    log(date(1382599077809));

    prints:

    "http://www.xillio.nl/"

  • Executing:

    log(date("2013-10-24T09:18:41.764"));

    prints:

    2013-10-24T09:18:41.764

  • Executing:

    log(date("24-10-2013, 09:18:41", "dd-MM-yyyy, hh:mm:ss"));

    prints:

    2013-10-24T09:18:41.0
