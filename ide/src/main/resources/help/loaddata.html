loaddata

loaddata (string uri, string encoding*)

Description

Returns a string variable containing the text of the local file uri. The optional parameter encoding sets the text encoding to use. It defaults to ‘UTF-8’. When encoding is set to to ‘BINARY’, the file is loaded into a binary variable, which can be then be used to e.g. calculate the md5.

Examples

  • Read a list of strings from a file, using loaddata and split:

    if (fileexists (file_uri)) {
    skiplist = global (split (loaddata (file_uri), "\r\n"));
    log ("Skipping: \n" + skiplist);
}

  • Read a webpage from a file and converting to node data type:

    cmspage = loaddata(cmsfile);
cmspage = stringtopage (cmspage);
